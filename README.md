Work time analysis tool.

This programme is not to force employees to work more.
This programme should help to analyse how to make breaks more effectively.
It also should help to improve performance and well-being by adjusting working time (begin, end and timing of breaks).


Capture time:

python3 clock.py --clock_in --> start working

python3 clock.py --break_start --> start of a break

python3 clock.py --break_end --> end of a break

python3 clock.py --clock_out --> end working


Afterwards the script will ask for "Lief?".
Here you can add numbers from 1-10 to document how you felt/worked this day.


To see correlation of each column with the "lief" performance/well-being indicator:
python3 clock.py --report
